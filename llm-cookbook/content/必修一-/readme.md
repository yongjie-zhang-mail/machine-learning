# 面向开发者的提示工程

本教程为吴恩达《ChatGPT Prompt Engineering for Developers》课程中文版，主要内容为指导开发者如何构建 Prompt 并基于 OpenAI API 构建新的、基于 LLM 的应用，包括：

- 书写 Prompt 的原则
- 文本总结（如总结用户评论）；
- 文本推断（如情感分类、主题提取）；
- 文本转换（如翻译、自动纠错）；
- 扩展（如书写邮件）
